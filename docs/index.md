## Getting Started

!!! tip "New to GitHub Copilot?"
    Start with the [Setup](setup.md) chapter to install and configure GitHub Copilot for your development environment.

!!! info "Prerequisites"
    - Basic programming knowledge (Python or C#)
    - GitHub account
    - Supported IDE (VS Code, Visual Studio, JetBrains IDEs, etc.)
    - GitHub Copilot subscription
