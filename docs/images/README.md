# Hero Images

This directory contains hero images for each chapter of the workshop. The images appear:
1. At the top of each chapter page as a hero banner
2. On the home page card tiles

## Required Images

Place your hero images here with these filenames:
- `setup-hero.jpg` - Setup chapter
- `deep-dive-hero.jpg` - Copilot Deep Dive chapter
- `practical-hero.jpg` - Practical Foundations chapter
- `best-practices-hero.jpg` - Best Practices chapter
- `advanced-hero.jpg` - Advanced Copilot chapter
- `context-hero.jpg` - Context Engineering chapter
- `project-hero.jpg` - Mini Project chapter

## Image Specifications

- **Dimensions**: 1200x400px (or similar 3:1 aspect ratio)
- **Format**: JPG or PNG
- **Style**: Should match the purple gradient theme (#667eea to #764ba2)
- **Content**: Relevant illustrations or visuals representing each chapter topic

## Fallback Behavior

If an image is not present, the card will display a gradient background instead, so the site will work even without custom images.

## Creating Hero Images

You can:
1. Use AI image generators (DALL-E, Midjourney, Stable Diffusion)
2. Create custom illustrations
3. Use stock photos with purple gradient overlays
4. Design graphics in tools like Figma, Canva, or Photoshop

**Tip**: Images similar to the anime-style illustrations in the Scaling Guacamole blog work great!
